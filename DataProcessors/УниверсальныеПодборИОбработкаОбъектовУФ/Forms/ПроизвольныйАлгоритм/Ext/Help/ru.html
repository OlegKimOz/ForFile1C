Обработка "Произвольный алгоритм".

Выполняет произвольный алгоритм на встроенном языке 1С: Предприятия 8.2.

Для выполнения произвольного алгоритма необходимо в поле редактирования ввести текст на встроенном языке и нажать кнопку "Выполнить" (клавиша F9). Заданный произвольный алгоритм будет выполнен последовательно для каждого объекта выборки, включенного в обработку.
Обращение к объекту из встроенного языка должно осуществляться посредством переменной "Объект".
Для регистров сведений тип переменной "Объект" представляет собой РегистрСведенийМенеджерЗаписи.

Другие доступные переменные и функции в алгоритме:

Ссылка - ссылка на обрабатываемый объект.
ИмяРегистра - имя регистра сведений.
СтрокаТаблицы - текущая строка таблицы значений выбранных объектов (только для регистров сведений).
СтруктураИзмерений - структура, содержащая имена измерений обрабатываемого регистра сведений.
СтруктураРесурсов - структура, содержащая имена ресурсов обрабатываемого регистра сведений.
СтруктураРеквизитов - структура, содержащая имена реквизитов обрабатываемого регистра сведений.
НаборЗаписей - набор записей регистра сведений с типом РегистрСведенийНаборЗаписей.

Функции:

// Функция определяет, существует ли табличная часть у объекта.
//
// Параметры
//  ИмяТЧ  – Строка - имя табличной части
//  МетаданныеОбъекта  – Объект описания метаданных - метаданные
//
// Возвращаемое значение:
//   Булево   – Истина, если существует реквизит.
//
Функция ЕстьТЧОбъекта(ИмяТЧ, МетаданныеОбъекта)

// Функция определяет, существует ли реквизит табличной части у объекта.
//
// Параметры
//  ИмяРеквизита  – Строка - имя реквизита
//  МетаданныеОбъекта  – Объект описания метаданных - метаданные
//  ИмяТЧ  – Строка - имя табличной части
//
// Возвращаемое значение:
//   Булево   – Истина, если существует реквизит.
//
Функция ЕстьРеквизитТЧОбъекта(ИмяРеквизита, МетаданныеОбъекта, ИмяТЧ)

// Функция определяет, существует ли реквизит у объекта.
//
// Параметры
//  Объект  – Любой объект метаданных, ссылка, переменная объектного типа, структура, строка таблицы значений и т.п
//  ИмяРеквизита  – Строка - имя реквизита
//
// Возвращаемое значение:
//   Булево   – Истина, если существует реквизит.
//
Функция ЕстьРеквизитОбъекта(Объект, ИмяРеквизита)

Кроме того, в произвольном алгоритме возможно применение экспортных процедур и функций общих модулей, системных переменных.

Пример написания произвольного алгоритма на встроенном языке:

Объект.Наименование = "Новое наименование";
Объект.Записать();
