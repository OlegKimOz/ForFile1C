Обработка предназначена для загрузки данных из файла в формате Microsoft Excel в информационную базу.

Возможные заголовки колонок задавать необходимо через запятую. При загрузке заголовки колонок из файла перед сравнением "сжимаются" (удаляются все пробелы, точки, запятые, точки с запятой; строка переводится в нижний регистр).
